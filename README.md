# blog

Hi,

I have created this blog project using python, django-framework, and front-end techs like html, css, and bootstrap 5.

operation of this project

Initially when a person opens this website he or she can view the blogs homepage and if they try to read more then the
page goes to login form, if user has an account then the user can login else they have to register.

after successfully logged in, then the user can read the blogs .
if user wants to post a new blog then they can create it by going to my post link in the navbar 

lastly user can also do crud operation that is create post, read post, update post and delete the post
